

【概念】:
    对象是一个实体(看得到，摸得着) 它可以是一个具体的人迪丽热巴也可以是一辆车...具体的车必须具体

【特点】
    所有的对象都有自己的属性(属于它的特性)和行为(可以做出的动作) 周星驰为例属性(身高，年龄... )行为(拍电影...)

【为什么要有对象？】
    我们知道保存一个值用变量 var num = 0 , 保存一串信息用数组 var arr = ['周润发', '迪丽热巴', 30, '周星驰']
    但是如果我们要保存一个人的全部信息呢? 一辆汽车的全部外观与性能呢? 就要用到对象了

【对象的创建及引用】
    利用字面量创建对象 里面的属性和方法我们采取键值对的方式来写Key: value 注意逗号隔开
    var person = {
        name: '张宇',   
        age: 21,
        sayHi: ()=>{
            console.log('你好')
        }
    } 
    //调用属性的两种方法
    console.log(person.name)
    console.log(person['age']);
    //调用方法的方法 
    console.log(person.sayHi());

【构造函数创建对象】
    原因:假设说我们要创建四大天王的对象的话 刘德华 张学友 黎明 郭富城四个他们的属性相同
    将公共(相同的)的部分抽取出来 假设相同的属性有姓名年龄等  相同的行为又有sing and ActingInMovies

【构造函数语法格式】 
    构造函数名第一个字母大写
    class Person{
        constructor(name){
            this.name = 属性值
        }
    } 
    调用语法 new 构造函数名()

【this】
    new的哪个具体对象就是指向的哪个
    var 刘德华 = new Star(); 这里我们new了一个刘德华this指向的具体对象就是刘德华 
    让刘德华这个对象有了name age sex这三个属性，值从哪里来？？ 
    属性值就是初始化的时候直接按照顺序传进去的(实参)传给(形参)

【构造函数与对象】
    构造函数指一大类 例如明星类 ， 汽车类 ， 水果类
    对象指的是一特类 例如刘德华 ， 法拉利 ， 火龙果

【new关键字】
    1， new 构造函数可以在内存中创建一个空的对象
    2， 构造函数中所有的this都去指向刚才创建对象
    3， 执行构造函数里面的代码为刚才创建的对象添加属性和方法
    4， 返回这个对象
    5， 所有构造对象必须使用 new 关键字实例化对象

【遍历对象】
    for(let k in obj )
    引用
    console.log(k)打印出的是属性名
    console.log(obj[k])打印的是具体值
